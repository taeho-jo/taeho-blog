## h2부터 사용

```ts {2,4} showLineNumbers
interface TEST_TYPE {
  name: string;
  age: number;
}
```

### 이유
---
* h1은 항상 제목에 표시되어있기 때문에, h1은 본문에서 사용하지 않습니다.

## 코드블럭 테스트

`야호`

- `rehype-code-titles`는 `:title=파일이름` 구문을 알아서 파싱해서 제목 바를 보여줘
- 위 구조가 정확히 지켜져야 작동해!

---

```ts:테스트1.mdx {3} showLineNumbers
interface User {
  name: string;
}
```

```ts:테스트2.mdx {2} showLineNumbers
interface TEST_TYPE {
  name: string
  age: number
}

```
